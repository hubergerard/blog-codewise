Problèmatique

Vous essayez de retirer un composant de votre application en supprimant les fichiers, mais une erreur apparaît dans le serveur :

Erreur Cannot find module

Alors comment faire ?

Supprimer via La CLI ?

Malheureusement non, il n'y a pas de commande pour supprimer un composant via la CLI et ce n'est toujours pas prévu pour les versions à venir.

La solution

Supprimer le dossier

Alors dans un premier temps, il faut supprimer le dossier du composant qui a été générer par la CLI si vous l'utilisez (ce qui est très recommandé).

Architecture du composant

Clic droit sur le dossier en question et sélectionner "supprimer".

Supprimer le dossier

Retirer la déclaration du module

Mais ce n'est pas terminé il vous reste une chose à faire encore si non le serveur vous affiche toujours cette erreur Cannot find module .

Il va falloir retirer deux lignes dans le fichier app.module.ts, l'import du composant en question et sa déclaration dans le NgModule :

app.module.ts

Conclusion

Si vous avez bien suivi vous n'aurez plus d'erreur dans votre serveur !
