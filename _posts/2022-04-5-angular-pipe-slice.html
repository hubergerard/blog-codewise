Pourquoi utiliser le pipe slice ?

Imaginons que vous ayez un aperçu d'article et que vous vouliez afficher une brève description sans pour autant surcharger la card de texte, ou alors que vous ayez un tableau d'objet et que vous souhaitez restreindre le nombre d'objet à l'affichage.

C'est à cela que sert le pipe slice et vous allez voir par la suite de cette article comment l'utiliser !

Que fait le pipe slice ?

Le pipe slice fait partie de l'API Angular qui appartient au CommonModule.

Dans son fonctionnement le pipe slice utilise l'API javascript Array.prototype.slice() et String.prototype.slice() pour réaliser son traitement.

Dans la documentation de l'API javascript il nous est dit que la méthode slice renvoi un tableau qui contient une copie d'une portion du tableau d'origine ou dans le cas d'une string il extrait la portion explicitée et la renvoi sans pour autant modifiée la chaîne originale.

Eh bien le pipe slice est un raccourci de ces méthodes javascript que vous pouvez implémenter dans vos templates !

Comment utiliser le pipe slice

Pour commencer voyons la syntaxe du pipe :

{% highlight ts linenos %} {% raw %}{{ myString | slice: 1 : 4 }}{% endraw %} {% endhighlight %}

On interpole la chaîne de caractère ou le tableau suivi d'une barre verticale et on mentionne le pipe slice suivi de deux points avec son paramètre start (obligatoire) et son paramètre end (optionnel).

Donc maintenant voyons comment l'utiliser selon les cas, pour cela il va falloir comprendre comment fonctionne ses paramètres.

Paramètre start : index de départ pour trancher le tableau ou la chaîne.

  • Si l'index de départ est positif le pipe renverra les éléments du tableau ou la chaîne à partir de l'index et à partir du début du tableau ou de la chaîne.
  • Si l'index de départ est négatif le pipe renverra les éléments ou la chaîne à partir de l'index et de la fin du tableau et de la chaîne.
  • Si l'index est positif et supérieur a la taille de la chaîne ou du tableau le pipe renverra vide !
  • Si l'index est négatif et supérieur à la taille de la chaîne ou du tableau alors le pipe renverra la chaîne ou le tableau complet.

Paramètre end : index de fin pour trancher la chaîne.

  • Si l'index de fin n'a pas été fourni alors le pipe renverra le tableau ou la chaîne jusqu'à la fin.
  • Si l'index de fin est positif, le pipe renverra tout avant l'index de fin et depuis le début.
  • Si l'index est négatif le pipe renverra les éléments avant l'index de fin et à partir de la fin du tableau ou de la chaîne.

Exemples

Avec un index de début et de fin positif :

{% highlight ts linenos %} myString:string = "abcdefghij"; {% endhighlight %}
slice.component.ts
{% highlight ts linenos %} {% raw %}{{ myString | slice: 1 : 4 }}{% endraw %} // Output : bcde {% endhighlight %}
slice.component.html

Avec un index de début et de fin négatif :

{% highlight ts linenos %} {% raw %}{{ myString | slice: 3 : -2 }}{% endraw %} // Output : defgh {% endhighlight %}
slice.component.html

Avec un index de début positif uniquement :

{% highlight ts linenos %} {% raw %}{{ myString | slice : 6 }}{% endraw %} // Output : fghij {% endhighlight %}
slice.component.html

Avec un index de départ négatif uniquement :

{% highlight ts linenos %} {% raw %}{{ myString | slice : -6 }}{% endraw %} // Output : efghij {% endhighlight %}
slice.component.html
