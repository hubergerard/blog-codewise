Qu'est-ce qu'Angular ?

Angular est un framework d'application web front, open-source basé sur le langage Typescript. Créer par Google, Angular est une réécriture complète de la même équipe qui à construit AngularJS.

Il permet de créer des applications clientes et dispose d'outils et de pratiques puissantes pour résoudre les problèmes de développement.

Avantages d'Angular

  • Développement multiplateforme
  • Haute qualité de l'application
  • Vitesse et performances améliorées
  • Processus de développement plus rapide
  • Code lisible et testable
  • Applications Web plus légères
  • Modèles efficaces de résolution de problèmes
  • Excellente bibliothèque de conception de matériaux

Cours Angular

Le guide complet d'Angular (anglais)

Ce cours s'adresse à tous. L'objectif est de vous apprendre les principes de base. Les directives, composants, modules, liaison de données, routage, HTTP, comment utiliser la CLI et vous présentera un projet réel.

  • Comment utiliser TypeScript
  • Fonctionnement de la liaison de données
  • L'architecture derrière une application Angular
  • Le routage et la gestion de la navigation
  • L'injection de dépendance
Guide complet Angular débutant (anglais)

Ce cours vous aidera à créer des applications à partir de zéro et maintenable pour en augmenter l'échelle.

  • Développement Web frontal
  • Compréhension approfondie des fondamentaux
  • Comprendre les applications à page unique
  • Ajout de la communication RESTAPI
Cours les bases d'Angular (français)

Dans ce cours, vous apprendrez rapidement à créer les composantes de base d'une application Angular. Vous apprendrez également à faire communiquer votre application Angular avec un serveur afin de créer une application web complète.

Cette formation est ouverte aux développeurs/architectes ayant des bases de Javascript, HTML et CSS.

Compétences acquises grâce à cette formation :

  • Appréhender la philosophie et le fonctionnement du framework Angular
  • Concevoir une application Angular avec TypeScript et RxJS
  • Créer vos propres composants Angular
  • Produire des applications robustes et efficaces
