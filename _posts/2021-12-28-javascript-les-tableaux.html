Définition

En informatique, un tableau (array en anglais) est une structure de données qui consiste en un ensemble d'éléments ordonnés accessibles par leur indice (ou index). C'est une structure de données de base que l'on retrouve dans chaque langage de programmation.

Déclaration

Il existe plusieurs manière de déclarer un tableau en JS :

  • La méthode la plus rapide et la plus utilisée est d’utiliser les “[ ]”.
  • Il est également possible de déclarer un tableau en utilisant le constructeur “Array”

Attention il y a une différence entre les deux méthodes.

{% highlight js linenos %} // Create an empty array const articles = []; // Create an empty array const shops = new Array(); // Create an array containing the value 5 const notes = [5]; console.log(notes); // Create an array with 5 empty elements const bulletin = new Array(5); console.log(bulletin); {% endhighlight %}

Accéder à un élément du tableau

Un tableau peut être déclaré avec plusieurs éléments séparés par des virgules.

On accède à un élément par son index.

Le 1er élément du tableau a pour index 0.

Accéder à l’élément à l’index 1 revient à récupérer le 2ème élément de notre tableau.

Si on accède à un index supérieur ou égale à la taille du tableau la valeur récupérée sera undefined.

{% highlight js linenos %} const cities = ["Berlin", "Lisbon", "Madrid", "Paris"]; console.log(cities[O]); // => Berlin console.log(cities[1]); // => Lisbon console.log(cities[4]); // => undefined {% endhighlight %}

Parcourir un tableau

On peut connaître la longueur d’un tableau en accédant à la propriété length.

On peut utiliser une boucle pour parcourir un tableau.

A chaque itération dans la boucle on utilise la variable compteur (i) comme valeur d’index pour accéder à l’élément.

{% highlight js linenos %} const cities = ["Berlin", "Lisbon", "Madrid", "Paris"]; console.log(cities.length); // => 4 for(let i = 0; i < cities.length; i++;){ console.log(cities[i]); } // expected output : // Berlin // Lisbon // Madrid // Paris {% endhighlight %}

Mais il y a aussi une autre manière de faire :

Le tableau appartient à une famille d’objetqu’on appelle : itérables

Les itérables peuvent être parcours avec une boucle for .. of

On déclare une variable (ex : city) qui va prendre tour à tour chaque valeur du tableau

{% highlight js linenos %} const cities = ["Berlin", "Lisbon", "Madrid", "Paris"]; for(const city of cities){ console.log(city); } // Berlin // Lisbon // Madrid // Paris {% endhighlight %}

À savoir

Nous pouvons mélanger plusieurs type de données dans un même tableau :

{% highlight js linenos %} const tab = ["Berlin", 12, false, Object]; {% endhighlight %}

Les tableaux en javascript sont muables

Qu'est-ce que veut dire muable ? c'est une chose qui peut être modifiable :

{% highlight js linenos %} const tab = [81, 12, 283, 29, 7]; console.log(tab = tab.sort((a, b) => a - b)); // expected output : [7, 12, 29, 81, 283] {% endhighlight %}

Dans cet exemple nous avons utilisé une méthode de tableau de mutateurs "sort()", biensûr il en existe d'autres et avec ceux-ci des méthodes de tableau non-mutateur, qui elles ne modifie pas le tableau mais en renvoie la représentation selon la méthode.
