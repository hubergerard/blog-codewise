Définition

Une structure de contrôle en informatique est une instruction particulière pouvant diriger le programme dans tel ou tel cas. Cela offre au programme plus de fléxibilité et de comportements.

Condition if/else

L’instruction if est une structure pour tester des conditions. Elle permet d'exécuter une série d'instructions si la condition est vérifiée. Grâce au mot-clé else nous pouvons spécifier des instructions dans le cas où la condition n’est pas vérifiée.

Exemple :

{% highlight js linenos %} const balance = 50; // 50 is greater than 0 => the condition is truthy. if (balance >= 0 { // Executes this statement console.log("Your balance is positive"); } else { // Do not execute this statement console.log("Your balance is negative"); } // expected output : Your balance is positive {% endhighlight %} {% highlight js linenos %} const balance = -100; // -100 is lesser than 0 => the condition is falsy. if (balance >= 0 { // Do not execute this statement console.log("Your balance is positive"); } else { // Executes this statement console.log("Your balance is negative"); } // expected output : Your balance is negative {% endhighlight %}

Switch case

La condition switch case permet de faire plusieurs test sur la même variable => gain de lisibilité et optimisation de code.

Cette instruction conditionnel simplifie le test de plusieurs valeurs d'une variable, car cette opération aurait été compliquée (mais faisable) avec des if imbriqués.

{% highlight js linenos %} const city = "Berlin"; // This evaluates city value switch(city) { case "Madrid": console.log("Spain is wild !"); break; // It matches this case case "Berlin": console.log("Germany is wild !"); break; case "Lisbon": console.log("Portugal is wild !"); break; default: console.log("This city is not wild yet !") } // expected output: Germany is wild ! {% endhighlight %} {% highlight js linenos %} const city = "New York"; // This evaluates city value switch(city) { case "Madrid": console.log("Spain is wild !"); break; case "Berlin": console.log("Germany is wild !"); break; case "Lisbon": console.log("Portugal is wild !"); break; // It does not match the cases default: console.log("This city is not wild yet !") } // expected output: This city is not wild yet ! {% endhighlight %}

Le mot-clé break est impératif après l'instruction pour stopper le switch si la condition est validée.

Boucle while

L’instruction while permet de spécifier un bloc d’instructions qui sera répété tant que la condition est vérifiée.

Attention, si la condition est toujours vérifiée le bloc d’instructions sera exécuté de façon infinie.

{% highlight js linenos %} // Initialize a counter i to 0 let i = 0; // Evaluates condition and executes statements // if condition is truthy while(i < 3) { console.log("This is a loop"); i = i + 1; } // When the condition is falsy it executes the next statement console.log("End of program"); // i value is 0, 0 < 3 is truthy // output : This is a loop // i value is 1, 1 < 3 is truthy // output : This is a loop // i value is 2, 2 < 3 is truthy // output : This is a loop // i value is 3, 3 < 3 is falsy // output : End of program {% endhighlight %}

Boucle do while

L’instruction do while se comporte comme la boucle while à l’exception que les instructions de la boucle seront exécutées au moins une fois avant de tester la condition.

{% highlight js linenos %} // Initialize a counter i to 0 let i = 0; // Executes statements do{ console.log("This is a loop"); // Increment i i = i + 1; } while(i < 3); // Evaluates the condition and executes statements // If the condition is truthy // When the condition is falsy it executes the next statement console.log("End of program"); // i is not tested the first time // output : This is a loop // i value is 1, 1 < 3 is truthy // output : This is a loop // i value is 2, 2 < 3 is truthy // output : This is a loop // i value is 3, 3 < 3 is falsy // output : End of program {% endhighlight %}

Boucle for

La boucle for à l'avantage de spécifier sur la même ligne séparée par des ;

  • L'initialisation du compteur
  • La condition à vérifier
  • l'incrémentation du compteur
{% highlight js linenos %} // Initialize a counter i to 0 // Evaluates the condition and executes statements // If condition is truthy for (let i = 0; i < 3; i++) { console.log("This is a loop"); } // Evaluates the condition and executes statements // If the condition is truthy // When the condition is falsy it executes the next statement console.log("End of program"); // i value is 0, 0 < 3 is truthy // output : This is a loop // i value is 1, 1 < 3 is truthy // output : This is a loop // i value is 2, 2 < 3 is truthy // output : This is a loop // i value is 3, 3 < 3 is falsy // output : End of program {% endhighlight %}
