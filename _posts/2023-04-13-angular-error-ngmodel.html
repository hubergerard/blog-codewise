Introduction

Si vous débuté le développement avec Angular, vous avez peut-être déjà rencontré l'erreur "Can't bind to 'ngModel' since it isn't a known property of 'input'" (Impossible de lier à 'ngModel' car ce n'est pas une propriété connue de 'input') lors de la création de formulaires.

Dans cet article, nous allons examiner ce que signifie cette erreur, pourquoi elle se produit et comment la résoudre.

Comprendre l'erreur

Cette erreur se produit généralement lorsque vous essayez de lier un modèle à un élément d'entrée dans un formulaire Angular.

Le modèle ngModel est utilisé pour lier les valeurs d'entrée à une variable dans le composant Angular correspondant.

Cela permet de récupérer facilement les données saisies par l'utilisateur et de les utiliser dans votre application.

Cependant, pour utiliser le modèle ngModel, vous devez d'abord importer le module FormsModule dans votre application Angular. Le module FormsModule fournit les directives et les pipes nécessaires pour créer des formulaires réactifs.

Si vous n'importe pas le module FormsModule, vous verrez l'erreur "Can't bind to 'ngModel' since it isn't a known property of 'input'". Cela signifie que le système ne reconnaît pas la propriété ngModel et ne peut donc pas la lier à l'élément d'entrée.

Résoudre l'erreur

Pour résoudre cette erreur, vous devez importer le module FormsModule dans votre application Angular. Vous pouvez le faire en ajoutant la ligne suivante à votre fichier app.module.ts :

{% highlight ts linenos %} import { FormsModule } from '@angular/forms'; {% endhighlight %}

Ensuite, vous devez ajouter FormsModule à la liste des imports dans la section @NgModule de votre fichier app.module.ts :

{% highlight ts linenos %} @NgModule({ imports: [ BrowserModule, FormsModule ], declarations: [ AppComponent ], bootstrap: [AppComponent] }) export class AppModule { } {% endhighlight %}

Une fois que vous avez importé le module FormsModule et que vous l'avez ajouté à la liste des imports, vous devriez être en mesure d'utiliser le modèle ngModel sans rencontrer l'erreur "Can't bind to 'ngModel' since it isn't a known property of 'input'".

Conclusion

L'erreur "Can't bind to 'ngModel' since it isn't a known property of 'input'" est une erreur courante que les développeurs Angular rencontrent lorsqu'ils créent des formulaires.

Cette erreur se produit généralement lorsque le module FormsModule n'est pas importé dans l'application Angular.

Pour résoudre cette erreur, vous devez importer le module FormsModule et l'ajouter à la liste des imports de votre fichier app.module.ts.
