Rechercher toutes les occurences

Pour renommer un projet Angular il va falloir trouver toutes les occurences dans les fichiers de l'ancien nom de projet pour pouvoir les remplacer.

Pour cela vous pouvez utilisez l'outil de recherche de Visual Studio Code :

Outil de recherche Visual Studio Code

Vous pouvez voir qu'il y a deux champs, le premier champ sert à rechercher l'expression, le second à remplacer l'expression chercher.

Remplacer donc toutes les occurences par le nouveau nom de projet que vous souhaitez grâce à cet outil.

Renommer le dossier du projet

Maintenant il vous faudra renommer le dossier du projet et c'est terminé vous pouvez lancez votre serveur !

Une erreur se produit ?

Vous lancez votre serveur et une erreur se produit ?

Supprimez le dossier node_modules , et lancez un npm install dans le terminal à la racine du dossier du projet et relancez le serveur.
