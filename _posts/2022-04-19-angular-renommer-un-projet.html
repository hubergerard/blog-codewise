Renommer un projet Angular est moins simple qu'il n'y parait : pas de commande pour renommer via la CLI, et des références au dossier dans divers fichiers.

Voici donc 3 étapes pour vous aider à renommer votre projet.

Rechercher toutes les occurences du nom à changer

Pour renommer un projet Angular, commencez par trouver toutes les occurences de l'ancien nom dans les fichiers du projet, pour pouvoir les remplacer.

Pour cela, vous pouvez utiliser l'outil "Search and replace" de Visual Studio Code (Ctrl + Shift + h) :

Outil de recherche Visual Studio Code

Vous pouvez voir qu'il y a deux champs, le premier champ sert à rechercher l'expression, le second à remplacer l'expression cherchée.

Remplacez donc toutes les occurences par le nouveau nom de projet que vous souhaitez grâce à cet outil.

Renommer le dossier du projet

Ensuite, renommez simplement le dossier du projet.

Réinstaller les dépendances

Certaines dépendances de votre projet Angular contiennent des références vers le dossier de votre projet. Il faut donc les réinstaller maintenant que tout a été renommé.

  • Supprimez le dossier node_modules
  • Lancez un npm install dans le terminal à la racine du dossier de projet.

C'est terminé, votre projet Angular est renommé ! Vous pouvez dès à présent relancer votre serveur pour vérifier que tout s'est bien passé.

Conclusion
